Github Pages无法访问的解决方法

背景

最近一年都在准备考研，没时间写博客，前几天终于结束了，心里也能放松一下了。不过，这几天一看自己的Github Pages竟然不能访问了，这让我很无奈，我这几天访问的时候就一直是下面这个鬼样子，给我气的啊，不过幸好找到解决办法了。┭┮﹏┭┮

解决办法

ping域名

自己打开命令行ping了一下自己的域名，结果找不到主机

修改首选DNS服务器的IP地址

依次点击控制面板、网络和Internet、网络和共享中心、WLAN、属性、Internet协议版本4（TCP/IPv4）、属性、使用下面的DNS服务器地址，并将其修改为114.114.114.114

如果上述方法不能仍然不能访问Github Pages，也可以将首选DNS的后两行改为223.5.5.5和223.6.6.6

再次ping域名

问题解决，开心到飞起O(∩_∩)O哈哈
