2.3 预编译

四部曲：

  1. 创建AO对象（即Activation Object，执行期上下文）
  2. 找形参和变量声明，将变量和形参名作为AO属性名，值为undefined
  3. 将实参值和形参统一
  4. 在函数体里面找函数声明，值赋予函数体

            d : f () {};
