MySQL服务启动

1 背景

今天舍友让我给他演示一下MySQL数据库的使用，前几天我不知道安装了什么软件，结果电脑感觉出了问题，连基本的ping命令都不能用了，所以就恢复系统，又查了一下网上的各种经验，终于搞定了♪(^∇^*)♪(^∇^*)

2 方法一

如图1所示，在桌面找到此电脑，右键点击管理

如图2所示，首先点击服务，然后右键MySQL80，并点击启动，即可启动MySQL服务。

【注】你的可能不是MySQL80，但开头只要是MySQL即可。

3 方法二

  1. 必须以管理员身份运行DOS，不然会提示服务名无效
  2. 必须进入到安装MySQL时的bin文件夹，里面还有个mysql.exe文件
  3. 在上述步骤的基础上，再运行命令net start mysql80

即可打开MySQL服务，进而可以使用sql语句通过MySQLWorkbench对数据库中数据进行操作

3 总结

数据库还是大二下半学期学的啊，现在才大三上半学期，都快忘了，在学习新知识的基础上，还是得不断复习以前学过的内容啊！！！
